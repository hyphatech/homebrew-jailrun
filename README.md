# Homebrew Tap: hyphatech/jailrun

This tap provides the Homebrew formula for **jailrun** (`jrun`) — run fully isolated environments on your machine, powered by FreeBSD.

- Upstream project: https://github.com/hyphatech/jailrun
- Package: https://pypi.org/project/jailrun/

## Install

```bash
brew tap hyphatech/jailrun
brew install jailrun
